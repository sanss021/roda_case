# Roda_case


Esse app é responsável por simular um armazenamento de vendas

## Acesse online
[Link](http://rodacase.herokuapp.com/)

## Requisitos

Python

Django (versão acima da 3.0+) 

Django Rest Framework 

## Instalação

1) Descompactar o arquivo zip

2) Acesse a pasta: cd roda_case

3) Instale o virtualenv:
```bash
pip install virtualenv / sudo apt-get install python3-venv
``` 
4) Ative o ambiente virtual que está na pasta 

```bash
source env/bin/activate
```
Caso tenha dificuldades crie um ambiente virtual com os seguintes pacotes:

```
pip install django
pip install djangorestframework
pip install django-filter
```


## Run

```python
python manage.py runserver
```
